# 🛍️ Customer Segmentation using Unsupervised Machine Learning

## 📌 Overview

Customer Segmentation is one of the most important applications of Unsupervised Machine Learning. This project leverages clustering techniques to group customers based on their purchasing behavior and characteristics. The identified customer segments help businesses create personalized marketing strategies, improve customer retention, and optimize business decisions.

---

## 🚀 Project Objective

The objective of this project is to:

- Analyze customer purchasing patterns
- Discover hidden customer groups
- Perform customer segmentation using K-Means Clustering
- Visualize customer clusters
- Generate business insights for targeted marketing

---

## 📊 Dataset

The dataset contains customer information such as:

- Customer ID
- Age
- Gender
- Annual Income
- Spending Score
- Other purchasing-related features

> **Note:** The dataset is used only for educational and analytical purposes.

## 📈 Results

The model successfully segments customers into meaningful groups based on purchasing behavior.

Example customer segments include:

- 💎 High Income – High Spending
- 💰 High Income – Low Spending
- 🎯 Average Income – Average Spending
- 📉 Low Income – High Spending
- 📦 Low Income – Low Spending

These insights can help businesses improve customer targeting and marketing strategies.
